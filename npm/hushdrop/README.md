# hushdrop

The `hush` command — drop anything (HTML, Markdown, PDF, file, or a whole site) and get a
branded, password-protected, zero-knowledge link on your own domain, in one step.

```bash
npx hushdrop report.html --managed   # zero-setup: publish to the managed tier (no account)
npx hushdrop report.html             # your own domain (after `hush init` + `hush deploy`)
```

Open-source · self-hosted · MIT. Site & docs: https://hushdrop.dev
